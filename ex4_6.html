Sensor de Gás

Introdução

Na atualidade, vemos um crescimento exponencial na quantidade de gases combustíveis na nossa atmosfera. Estar exposto, em abundância, a esses gases pode-se gerar malefícios a sua saúde, por isso, foi desenvolvido este experimento que busca detectar a quantidade de gás presente e alertar quando passar do previsto no código.

Componentes

1 - Arduino Uno R3;

1 - LED;

1 - Resistor;

1 - Sensor MQ-5;

1 - Buzzer;

7 - Jumpers.

Código

#define ledPin 7
#define sensor A0
int gas_value;
void setup()
{
pinMode(sensor, INPUT);
pinMode(ledPin, OUTPUT);
Serial.begin(9600);
Serial.println("MQ5 Heating Up!");
delay(20000);
}
void loop()
{
gas_value = analogRead(sensor);
Serial.print("Sensor Value:");
Serial.println(gas_value);
if (gas_value > 250) {
digitalWrite(ledPin, HIGH);
}
else{
digitalWrite(ledPin, LOW);
}
}
