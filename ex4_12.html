Sensor de cores

1.1 Introdução


Um sensor de cor pode ter a utilidade de identificara cor de um objeto, é um dispositivo muito simples e fácil de usar. Para realizar esse experimento os componetes são: sensor Cor TCS3200 e o Arduino UNO

Código:

/*
Pino S0 do Sensor -> Pino D8 do Arduino
Pino S1 do sensor -> Pino D7 do Arduino
Pino S2 do Sensor -> Pino D6 do Arduino
Pino S3 do Sensor -> Pino D5 do Arduino
Pino OUT do sensor -> Pino D4 do Arduino
*/
#define S0_PIN 5
#define S1_PIN 4
#define S2_PIN 7
#define S3_PIN 6
#define OUT_PIN 8
void setup()
{
// Define os pinos S0, S1, S2, S3 como saída
pinMode(S0_PIN, OUTPUT);
pinMode(S1_PIN, OUTPUT);
pinMode(S2_PIN, OUTPUT);
pinMode(S3_PIN, OUTPUT);
//Definir OUT_PIN como entrada
pinMode(OUT_PIN, INPUT);
// Definir escala de largura de pulso para 20%
digitalWrite(S0_PIN, ALTO);
digitalWrite(S1_PIN, BAIXO);
// Ativar UART para depuração
Serial.begin(9600);
}
