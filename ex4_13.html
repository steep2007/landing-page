Sistema de irrigação

1.1 Introdução


Muitas pessoas gostam de ter plantas em suas casas, as vezes a pessoa precisa fazer uma viagem por exemplo deixando a planta sem o seu solo umido, podendo murchar a ou até mesmo matar a plantinha. Para a solução desse problema temos o sistem de irregação automatico. utilizando os sseguintes componentes:

• Sensor de umidade

• Módulo de relé 5v

•6v Mini bomba de água com tubo pequeno

• Fios de conexão

• Bateria 5v

int soloMoistureValue = 0; int porcentagem=0;

}
void loop() {
soloMoistureValue = analogRead(A0);
Serial.println(porcentagem);
porcentagem = map(soilMoistureValue, 490, 1023, 100, 0);
if(porcentagem < 10)
{
Serial.println("bomba ligada");
digitalWrite(3,BAIXO);
}
if(porcentagem >80)
{
Serial.println("bomba desligada");
digitalWrite(3,HIGH);
}
}
